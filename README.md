First, extract the file if you download as zip.
- If you have the Geode texture loader, simply place the extracted folder in the packs folder, by clicking the folder icon in the textures menu in the game
- If you are vanilla, navigate to Windows C. > Program Files (x86) > Steam > steamapps > common > Geometry Dash > Resources, then paste the GJ_gamesheet02-uhd.png file from the folder and click on "Replace existing files"
